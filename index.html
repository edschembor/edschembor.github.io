Welcome to my personal page.

I'm Edward Schembor, a senior Computer Science major at Johns Hopkins University. My interests currently lie in distributed systems, machine learning, and computer engineering in general. I will be working at Microsoft working on Azure beginning summer 2016.

This is mainly a place for me to organize my thoughts, projects, etc. Below is a list of things that I've created. For a more complete list, check my Github profile.

SVM For Loan Default Predictions: A machine learning paper based on using a dual-from Support Vector Machine in order to predict Loan Defaults on LendingClub loan data.

PrimalPy: Python package for prime numbers. Currently a collection of primality tests - prime generation algorithms in the works. It originated as a way for me to practice translating math into efficient code.

HAChat: A highly available distributed chat application - clients can continue to chat even when servers become partitioned as the system is eventually consistent. Uses lamport timestamps for merge conflicts.

Ring Protocol: A C implementation of the token ring protocol for distributed system file transfers.

Notester: Note-sharing site for AP classes and the SATs. My first trial in web design. Site has been taken down due to lack of funding but am working on getting it back up soon.
