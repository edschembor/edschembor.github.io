Using OpenCV to Tell If My Laundry Is Done

The dryer in my house has a setting where it continues until your clothes are a certain amount dry, as opposed to a hard time limit. Because of this, I never know when my laundry is done and end up unnecessarily checking the machine multiple time. So, I decided to dabble in computer vision and see if I could set up a webcam to watch the machine's dial and see if it had finished.

OpenCV is an opensource computer vision library, with C++, C, Python, and Java interfaces. It also has a pretty good tutorial/documentation community.
